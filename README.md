# 电信服务开户界面示例

这是一个电信服务商开户界面的学习练习示例，不含任何真实品牌。

## 功能特性

- ✅ 用户信息表单（姓名、证件、联系方式等）
- ✅ 套餐选择（基础/标准/尊享三种套餐）
- ✅ 号码选择（吉祥号/普通号筛选）
- ✅ 信息确认与提交
- ✅ 响应式设计，支持移动端
- ✅ 美观的 UI 界面
- ✅ 分步骤表单引导

## 技术栈

- HTML5
- CSS3（自定义变量、Flexbox、Grid）
- 原生 JavaScript（无框架依赖）

## 本地运行

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 文件
3. 或使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve
   ```

## 使用说明

1. **步骤 1 - 用户信息**：填写姓名、证件类型及号码、联系电话等基本信息
2. **步骤 2 - 选择套餐**：从三种套餐中选择适合的服务方案
3. **步骤 3 - 选择号码**：浏览并选择心仪的手机号码
4. **步骤 4 - 确认提交**：核对信息并提交申请
